# copilot-capstone
IT Support Copilot Automation Agent using Microsoft Copilot Studio

https://m365.cloud.microsoft/chat/?titleId=T_ac3be8e5-0a26-b69e-bcc1-0928847c27b5&source=embedded-builder

## Overview
The IT Support Copilot Agent is a virtual assistant designed to help users resolve common Tier-1 technical issues efficiently. It simulates a real-world IT support helpdesk by guiding users through troubleshooting steps, classifying tickets, and escalating issues when necessary.

---

## Objectives
- Provide step-by-step troubleshooting support
- Improve user experience in IT helpdesk scenarios
- Simulate real-world IT support workflows
- Automate ticket classification and escalation

---

## 🧩 Troubleshooting Workflows Included
### 1. VPN Not Connecting
- Date/time check
- Internet check
- Password and profile verification
- Escalation to Network Team

### 2. Email / Outlook Issue
- Internet validation
- Error message checks
- Mailbox and sync troubleshooting
- Escalation to Email Support Team

### 3. Slow Computer
- Storage check
- Task manager review
- Antivirus scan guidance
- Escalation to Hardware Support

### 4. Wi-Fi / Network Issue
- Router restart and connectivity checks
- IP refresh troubleshooting
- Escalation for outages or repeated failures

## Example Use Cases

### Network Issue
- User cannot connect to Wi-Fi  
- Agent guides through connectivity checks  
- Classifies as **Network | Medium Priority**

### Login Problem
- User unable to log in  
- Agent checks credentials, password reset  
- Escalates if account is locked  

### Email Issue
- Email not syncing  
- Agent verifies settings and connectivity  

---

## How It Works

1. User describes the issue  
2. Copilot asks clarifying questions  
3. Provides guided troubleshooting  
4. Classifies the ticket  
5. Resolves or escalates
