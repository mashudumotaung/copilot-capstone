# copilot-capstone
IT Support Copilot Automation Agent using Microsoft Copilot Studio

---

## 🧩 Troubleshooting Workflows Included
### 1. VPN Not Connecting
- Date/time check
- Internet check
- Password and profile verification
- Escalation to Network Team

### 2. Email / Outlook Issue
- Internet validation
- Error message checks
- Mailbox and sync troubleshooting
- Escalation to Email Support Team

### 3. Slow Computer
- Storage check
- Task manager review
- Antivirus scan guidance
- Escalation to Hardware Support

### 4. Wi-Fi / Network Issue
- Router restart and connectivity checks
- IP refresh troubleshooting
- Escalation for outages or repeated failures

---

##  Knowledge Base (SOP Documents)
The Copilot agent uses internal SOP documents for RAG-style responses, including:
- Password Reset & Account Unlock
- VPN Setup & Connectivity
- Email / Outlook Troubleshooting
- Printer Setup & Troubleshooting
- Slow Computer Troubleshooting
- Wi-Fi / Network Troubleshooting

##  Testing Summary
The agent was tested using common Tier-1 scenarios such as:
- VPN failing to authenticate
- Outlook unable to send or receive emails
- Slow device performance
- Printer offline issues
- No internet connectivity

Testing confirmed:
- Correct workflow branching (IF/ELSE logic)
- Accurate SOP-based responses
- Proper ticket classification output
- Successful escalation summary generation
