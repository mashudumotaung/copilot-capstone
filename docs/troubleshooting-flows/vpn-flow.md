# VPN Troubleshooting Flow

## Flow Steps
1. User reports VPN issue
2. Copilot checks system date/time
3. Copilot checks internet connection
4. Copilot checks password/login
5. Copilot suggests VPN profile reset
6. If unresolved, escalate to Network Team
