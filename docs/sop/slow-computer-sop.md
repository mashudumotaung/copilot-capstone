# SOP 05: Slow Computer Troubleshooting

**Document Title:** SOP – Slow Computer Troubleshooting  
**Document Code:** SOP-PERF-005  
**Version:** 1.0  
**Owner:** IT Support Team  
**Last Updated:** 2026-04-14  

---

## 1. Purpose
To diagnose and resolve performance issues on user computers.

---

## 2. Scope
Applies to all devices experiencing:
- Slow performance  
- Freezing or lagging  
- Long startup times  
- Applications not responding  

---

## 3. Requirements
- Admin access (if needed)  
- Antivirus software installed  
- System tools available  

---

## 4. Common Symptoms
- Computer slow to boot  
- Programs take long to open  
- System freezing  
- High CPU or memory usage  

---

## 5. Procedure

### 5.1 Basic Checks
- Restart the computer  
- Close unnecessary applications  
- Check internet connection (if cloud apps are slow)  

---

### 5.2 Check Storage
- Open **This PC**  
- Check available disk space  
- If low (<15%), delete unnecessary files  

---

### 5.3 Task Manager Check
- Press **Ctrl + Shift + Esc**  
- Identify high CPU/memory apps  
- End unnecessary tasks  

---

### 5.4 Malware Scan
- Run full antivirus scan  
- Remove detected threats  

---

### 5.5 System Optimization
- Disable startup programs  
- Clear temporary files  
- Update operating system  

---

## 6. Escalation
Escalate to IT Support if:
- Hardware failure suspected  
- Persistent slowness after cleanup  
- System crashes occur  

---

## 7. Ticket Classification
- Category: Performance / Hardware  
- Subcategory: Slow Computer  
- Priority: Medium (High if productivity affected)

---

## 8. Technician Notes
System optimized and performance improved after troubleshooting steps.
