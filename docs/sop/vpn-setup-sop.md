# SOP 02: VPN Setup & Connectivity

**Document Title:** SOP – VPN Setup & Connectivity  
**Document Code:** SOP-NET-002  
**Version:** 1.0  
**Owner:** IT Network Team  
**Last Updated:** 2026-04-14  

---

## 1. Purpose
To guide users in installing, configuring, and troubleshooting VPN access to the company network.

---

## 2. Scope
Applies to all users who need remote access to company systems via VPN.

---

## 3. Requirements
- Active company account  
- VPN client installed  
- Internet connection  

---

## 4. Common Issues
- VPN not connecting  
- Authentication failure  
- Slow VPN connection  
- Cannot access internal systems  

---

## 5. Procedure

### 5.1 Install VPN Client
- Download approved VPN software  
- Install using default settings  
- Restart device if required  

---

### 5.2 Login to VPN
- Open VPN application  
- Enter company username and password  
- Complete MFA verification if required  

---

### 5.3 Connect to Network
- Select company server/location  
- Click Connect  
- Wait for confirmation message  

---

### 5.4 Troubleshooting
If VPN fails:
- Check internet connection  
- Restart VPN application  
- Re-enter credentials  
- Disable firewall temporarily (if allowed)  

---

## 6. Escalation
Escalate to Network Team if:
- VPN repeatedly fails to connect  
- Authentication errors persist  
- Server not reachable  

---

## 7. Ticket Classification
- Category: Network  
- Subcategory: VPN Access  
- Priority: High (if remote work is blocked)  

---

## 8. Technician Notes
VPN access configured or restored successfully. User able to connect to company network.
