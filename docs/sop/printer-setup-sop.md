# SOP 04: Printer Setup & Troubleshooting

**Document Title:** SOP – Printer Setup & Troubleshooting  
**Document Code:** SOP-HW-004  
**Version:** 1.0  
**Owner:** IT Service Desk  
**Last Updated:** 2026-04-14  

---

## 1. Purpose
To provide steps for installing, configuring, and troubleshooting printer-related issues.

---

## 2. Scope
Applies to all users who experience:
- Printer not printing  
- Printer offline  
- Cannot add printer  
- Paper jam / printing errors  
- Wrong printer selected  

---

## 3. Requirements
- Printer connected to network or USB  
- User device connected to company network  
- Printer drivers available (if required)

---

## 4. Common Symptoms
- “Printer offline”  
- Print jobs stuck in queue  
- Printer not found  
- Printer prints blank pages  
- Printer shows error lights  

---

## 5. Procedure

### 5.1 Check Printer Status
- Confirm printer has power  
- Confirm printer is connected (Wi-Fi/Network cable/USB)  
- Check printer display for error messages  

---

### 5.2 Restart Printer and PC
- Turn off printer for 10 seconds  
- Turn printer back on  
- Restart computer  

---

### 5.3 Add Printer (Windows)
- Go to **Settings → Devices → Printers & scanners**  
- Click **Add a printer or scanner**  
- Select the printer from list  
- If not found, select **The printer that I want isn’t listed**  

---

### 5.4 Clear Print Queue
- Open **Control Panel → Devices and Printers**  
- Right-click printer → **See what’s printing**  
- Cancel stuck print jobs  
- Restart Print Spooler service if needed  

---

### 5.5 Driver Installation (If required)
- Download approved driver from vendor or internal IT share  
- Install driver  
- Restart device  
- Test print  

---

## 6. Escalation
Escalate to Hardware/Support Team if:
- Printer hardware failure  
- Printer not detected on network  
- Driver installation fails  
- Printer requires replacement  

---

## 7. Ticket Classification
- Category: Hardware  
- Subcategory: Printer Setup / Printer Not Printing  
- Priority: Low to Medium (High if business-critical printing required)

---

## 8. Technician Notes
Printer configured or restored. Print queue cleared and test page printed successfully.
