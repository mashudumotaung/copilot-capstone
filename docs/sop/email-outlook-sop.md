# SOP 03: Email & Outlook Troubleshooting

**Document Title:** SOP – Email & Outlook Troubleshooting  
**Document Code:** SOP-COMM-003  
**Version:** 1.0  
**Owner:** IT Service Desk  
**Last Updated:** 2026-04-14  

---

## 1. Purpose
To resolve common email and Outlook issues affecting user communication and productivity.

---

## 2. Scope
Applies to all users experiencing issues with:
- Sending emails  
- Receiving emails  
- Outlook login problems  
- Outlook not opening  

---

## 3. Requirements
- Active internet connection  
- Valid company email account  
- Outlook application installed  

---

## 4. Common Issues
- Emails not sending  
- Emails not received  
- Outlook login failure  
- Sync errors  
- Mailbox full  

---

## 5. Procedure

### 5.1 Basic Checks
- Verify internet connection  
- Restart Outlook  
- Check if other websites load  

---

### 5.2 Login Issues
- Confirm username and password  
- Reset password if required  
- Re-login to Outlook  

---

### 5.3 Sending/Receiving Issues
- Check “Outbox” for stuck emails  
- Ensure mailbox is not full  
- Remove large attachments  
- Refresh sync settings  

---

### 5.4 Advanced Fix
- Remove and re-add email account  
- Repair Outlook application  
- Update Outlook version  

---

## 6. Escalation
Escalate to Email/Exchange team if:
- Mailbox sync fails repeatedly  
- Account authentication issues persist  
- Server connection errors occur  

---

## 7. Ticket Classification
- Category: Software / Communication  
- Subcategory: Email / Outlook  
- Priority: Medium (High if communication is blocked)  

---

## 8. Technician Notes
Email issue investigated and resolved. User able to send and receive emails after troubleshooting steps.
